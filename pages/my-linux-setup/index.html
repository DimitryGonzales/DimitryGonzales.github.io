About This Guide
This guide is based on Arch Linux, using its structure, tools and conventions as reference. While many steps may apply to other distributions, some commands or configurations are specific to Arch and its derivatives.

fstrim

Wiki

About:

fstrim is used on a mounted filesystem to discard (or "trim") blocks which are not in use by the filesystem. This is useful for solid-state drives (SSDs) and thinly-provisioned storage.

Actions:

  1. If the system is on a SSD:
    1. Enable fstrim.timer service

Pacman

Wiki

About:

The pacman package manager is one of the major distinguishing features of Arch Linux. It combines a simple binary package format with an easy-to-use Arch build system. The goal of pacman is to make it possible to easily manage packages, whether they are from the official repositories or the user's own builds.

Packages:

pacman-contrib
sudo pacman -S --needed pacman-contrib

Actions:

  1. Edit /etc/pacman.conf
    1. In Misc options:
      1. Uncheck Color
      2. Uncheck VerbosePkgLists
      3. Change ParallelDownloads to 10
    2. In Repositories:
      1. Uncheck [multilib]
  2. Enable and Start paccache.timer service
systemctl enable --now paccache.timer

PipeWire

Wiki

About:

PipeWire is a project that aims to greatly improve handling of audio and video under Linux. It provides a low-latency, graph-based processing engine on top of audio and video devices that can be used to support the use cases currently handled by both PulseAudio and JACK.

Hyprland

Wiki

About:

Hyprland is an independent tiling Wayland compositor written in C++. Noteworthy features of Hyprland include dynamic tiling, tabbed windows, a clean and readable C++ code-base, and a custom renderer that provides window animations, rounded corners, and Dual-Kawase Blur on transparent windows.

Packages:

sudo pacman -S --needed hyprland hyprpolkitagent hyprlock hyprshot hyprpicker xdg-desktop-portal xdg-desktop-portal-gtk xdg-desktop-portal-hyprland

Greetd

Wiki

About:

Greetd is a minimal, agnostic and flexible login manager daemon which does not make assumptions about what the user wants to launch, should it be console-based or graphical. Any script or program which can be started from the console may be launched by greetd, which makes it particularly suitable for Wayland compositors. It can also launch a greeter to start user sessions, like any other display manager.

Packages:

greetd greetd-tuigreet
sudo pacman -S --needed greetd greetd-tuigreet

Actions:

  1. Edit /etc/greetd/config.toml:
    1. Change command to tuigreet --cmd Hyprland
