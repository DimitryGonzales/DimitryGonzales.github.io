Pacman

Wiki

About:

The pacman package manager is one of the major distinguishing features of Arch Linux. It combines a simple binary package format with an easy-to-use Arch build system. The goal of pacman is to make it possible to easily manage packages, whether they are from the official repositories or the user's own builds.

Packages:

pacman-contrib
sudo pacman -S --needed pacman-contrib

Actions:

  1. Edit /etc/pacman.conf
    1. In Misc options:
      1. Uncheck Color
      2. Uncheck VerbosePkgLists
      3. Change ParallelDownloads to 10
    2. In Repositories:
      1. Uncheck [multilib]
  2. Enable and Start paccache.timer service
systemctl enable --now paccache.timer

Pipewire

Wiki

About:

PipeWire is a project that aims to greatly improve handling of audio and video under Linux. It provides a low-latency, graph-based processing engine on top of audio and video devices that can be used to support the use cases currently handled by both PulseAudio and JACK.

Packages:

pipewire lib32-pipewire wireplumber helvum pavucontrol pipewire-audio pipewire-alsa pipewire-pulse pipewire-jack lib32-pipewire-jack
sudo pacman -S --needed pipewire lib32-pipewire wireplumber helvum pavucontrol pipewire-audio pipewire-alsa pipewire-pulse pipewire-jack lib32-pipewire-jack
